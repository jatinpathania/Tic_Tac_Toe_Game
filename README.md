## ğŸ¯ How to Play

1. The game board consists of a 3x3 grid
2. Two players take turns clicking on empty cells
3. Player 1 uses "X" and Player 2 uses "O"
4. **Win Condition**: Get three of your marks in a row (horizontal, vertical, or diagonal)
5. **Draw**: If all 9 cells are filled with no winner, the game ends in a draw
6. Click the "Restart" button to play again
